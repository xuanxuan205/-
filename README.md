## ❓ 常见问题

<details>
<summary><strong>🔍 程序无法启动？</strong></summary>

**解决方案**:
- 检查是否安装了 .NET Framework 4.7.2+
- 尝试以管理员身份运行程序
- 确认系统满足最低配置要求
- 检查杀毒软件是否误报拦截

</details>

<details>
<summary><strong>⚡ 破解速度很慢？</strong></summary>

**优化建议**:
- 检查CPU使用率，避免过热降频
- 调整线程数设置（建议CPU核心数×2）
- 关闭其他占用资源的程序
- 使用SSD硬盘提升IO性能

</details>

<details>
<summary><strong>🔐 破解不成功？</strong></summary>

**可能原因**:
- 密码确实很复杂，需要更长时间
- 尝试切换不同的破解模式
- 检查密码长度和字符集设置
- 考虑使用更大的字典文件

</details>

<details>
<summary><strong>🛡️ 杀毒软件报毒？</strong></summary>

**说明**:
- 这是破解类工具的常见误报
- 程序完全安全，无任何恶意代码
